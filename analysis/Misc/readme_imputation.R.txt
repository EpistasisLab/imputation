### Descriptions of R scripts used in imputation analysis

plotImputationGWAS.R
	script for generating plots of GWAS results
missingnessPlots.R
	script for generating plots to summarize the patterns of missingness the EHR
EHRsummary.R
	script for generating plots to summarize the data in the EHR
testNormPredForGWAS.R
	script generating the imputed dataset used in GWAS_test1
evaluatingResults.R
	script for generating plots of error rates for each imputation method 
miceImputeAllMethods.R
	script to run a variety of imputation methods from the MICE package
miceImputeNormPredictParamters.R
	script to test paramter ranges using the mice norm.predict method
miceImputePMMparameters.R
	script to test paramter ranges using the mice pmm method
